Claim boundaries:

- the smoke result is synthetic/small-canonical evidence;
- `launch_block` can mean first observed rather than true commercial launch;
- future audit evaluates the frozen release catalog;
- simulation and OPE are controlled laboratories, not production causal lift;
- the immutable release store is local same-filesystem coordination, not distributed consensus;
- GitHub attestations are configured but exist only after a hosted release workflow executes.
