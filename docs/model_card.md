# Model Card v0.6.0

## Intended use

Offline research, portfolio evidence, controlled product-search serving, and local release qualification. Not intended for autonomous production launch or unreviewed business decisions.

## Champion

- block-local BM25 and dense retrieval;
- frozen release-catalog hybrid retrieval;
- native-JSON LambdaMART;
- temporally calibrated behavioral model;
- global Q-RSBT transfer;
- bounded coverage-overreach gate.

## Main risks

- substitute relation error and domain shift;
- position-biased interactions;
- sparse relevance coverage;
- first-observed time mistaken for true launch;
- cold promotion displacing warm products;
- feedback loops and simulation misspecification;
- artifact, code, dependency, generation, or pointer skew;
- worker saturation and multi-process capacity misunderstanding.

## Controls

- strict temporal retrieval and no-oracle inference;
- calibration and future-block gates;
- global substitute support and uncertainty shrinkage;
- bounded promotion and non-inferiority constraints;
- multi-replication lower-tail dynamic gates;
- known-propensity OPE laboratory;
- hash-before-load and source/config/model fingerprints;
- immutable generation publication and revalidated rollback;
- per-worker admission control and 503 overload response;
- real one-worker and two-worker Uvicorn validation.
